# Guide utilisateur - FootLink

Date : 15 mai 2026

## 1. Visiteur

### Voir les clubs

1. Aller sur la page d'accueil.
2. Cliquer sur `Clubs`.
3. Rechercher un club par nom, ville ou niveau.
4. Ouvrir une fiche club pour voir ses stats, effectif, matchs et galerie.

### Voir un joueur

1. Depuis la page d'accueil ou une fiche club, cliquer sur un joueur.
2. Consulter sa photo, son poste, ses stats et sa bio.

## 2. Joueur

### Creer un compte joueur

1. Cliquer sur `Cr├®er mon profil joueur`.
2. Remplir les informations : nom, email, mot de passe, poste, age, ville, pied fort.
3. Choisir un club a rejoindre ou cocher `Je n'ai pas d'├®quipe`.
4. Valider l'inscription.
5. Verifier l'adresse email via le lien recu.
6. Se connecter.

### Modifier son profil

1. Ouvrir l'espace joueur.
2. Aller dans `Profil`.
3. Modifier les informations.
4. Cliquer sur `Sauvegarder`.

### Ajouter une photo

1. Ouvrir l'espace joueur.
2. Cliquer sur `Changer la photo`.
3. Choisir une image.
4. Attendre la fin du televersement.

### Rejoindre un club

1. Aller dans la section `Mes clubs`.
2. Choisir un club.
3. Cliquer sur `Demander ├Ā rejoindre`.
4. Attendre l'acceptation du club.

### Repondre a une invitation

1. Aller dans `Invitations`.
2. Cliquer sur `Accepter` ou `Refuser`.

## 3. Club

### Creer un compte club

1. Cliquer sur `Inscrire mon club`.
2. Remplir les informations du responsable et du club.
3. Verifier l'adresse email.
4. Se connecter au dashboard equipe.

### Gerer l'identite du club

1. Ouvrir l'espace equipe.
2. Aller dans `Club`.
3. Ajouter ou changer le logo.

### Gerer la galerie

1. Aller dans `Galerie`.
2. Ajouter une legende.
3. Cliquer sur `Ajouter une photo`.
4. Les photos supplementaires restent accessibles dans une liste deroulante.

### Gerer les demandes

1. Aller dans `Demandes`.
2. Accepter ou refuser les demandes recues.

### Inviter un joueur

1. Aller dans `Recherche`.
2. Filtrer par position, ville ou age.
3. Cliquer sur `Inviter`.

### Gerer l'effectif

1. Aller dans `Joueurs`.
2. Modifier le role club.
3. Retirer un joueur si necessaire.

### Ajouter un match et des stats

1. Aller dans `Matchs`.
2. Creer un match.
3. Choisir un joueur et un match.
4. Ajouter buts, passes, cartons et homme du match.
5. Valider.

## 4. Mot de passe oublie

1. Cliquer sur `Mot de passe oubli├®`.
2. Entrer l'email du compte.
3. Ouvrir le lien recu par email.
4. Definir un nouveau mot de passe.

## 5. Bonnes pratiques

- Utiliser une vraie adresse email.
- Ajouter une photo nette.
- Garder les stats a jour.
- Eviter les descriptions trop longues.
- Verifier les demandes en attente regulierement.
